Manual do Angular

Introdução ao Angular

O Angular é uma estrutura de desenvolvimento de aplicativos front-end baseada em TypeScript.

Versões

A versão mais recente do Angular é a X.Y.Z. Verifique o site oficial do Angular para obter informações atualizadas sobre versões.

Instalando o Angular CLI

O Angular CLI (Command Line Interface) é uma ferramenta que simplifica a criação, desenvolvimento e compilação de projetos Angular.

npm install -g @angular/cli

Verificando a instalação

Verifique se o Angular CLI foi instalado corretamente:

ng --version

Criando um projeto em Angular

Use o Angular CLI para criar um novo projeto:

ng new nome-do-projeto

Executando aplicações em Angular

Entre no diretório do projeto e inicie a aplicação:

cd nome-do-projeto
ng serve

Acesse a aplicação em http://localhost:4200/.

Conclusão

Este é um guia básico para começar com o Angular. Consulte a documentação oficial do Angular para aprender mais.
