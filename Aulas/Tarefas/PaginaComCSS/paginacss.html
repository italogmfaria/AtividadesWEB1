Introdução ao CSS

   O CSS(Cascading Style Sheets) é um documento que é empregado para estilizar o conteúdo de um documento HTML, ou seja, controlar a apresentação de documentos de marcação. O HTML controla a estrutura de uma página, ao passo que o CSS determina sua aparência. Uma folha de estilos é uma coleção de regras de formatação aplicadas a um documento HTML.

Versões
  • CSS1 (Cascading Style Sheets nível 1):

    Foi a primeira versão do CSS e introduziu o conceito de separação de conteúdo e apresentação. Suportava estilos básicos, como fontes, cores, margens e alinhamentos. Limitações significativas em comparação com versões posteriores.

  • CSS2 (Cascading Style Sheets nível 2):

    Lançado em várias partes entre Introduziu recursos mais avançados, como posicionamento de elementos, seletores avançados, efeitos de transição e folhas de estilo para mídia específica. Fornecia mais controle sobre a formatação e layout.

  • CSS2.1 (Cascading Style Sheets nível 2.1):

    Uma revisão do CSS2 que corrigia erros e inconsistências na especificação original. Tornou-se uma recomendação do W3C em 2011.

  • CSS3 (Cascading Style Sheets nível 3):

    Introduzido a partir de 1999, mas ainda em desenvolvimento. Dividido em módulos, cada um adicionando novos recursos, como sombras, gradientes, bordas arredondadas, transformações, animações, entre outros. Permitiu maior criatividade e flexibilidade no design web. Não é uma única especificação, mas um conjunto de módulos independentes. As novas funcionalidades ainda estam sendo adicionadas como módulos ao CSS3.

Como aplicar o CSS

   As regras do CSS podem ser aplicadas de três formas distintas:

  • inline: estilos aplicados diretamente em um elemento HTML, dentro do documento
  • incorporado: O elemento style permite aplicar várias regras de uma vez no próprio documento HTML.
  • folha de estilos externa: um documento com extensão .css permite aplicar regras a um website inteiro de uma vez.

   Os estilos são aplicados na ordem em que são encontrados, cascateando dos estilos externos até os estilos inline.

Sintaxe em CSS

   Regra CSS é a unidade básica de construção de uma folha de estilos. Sintaxe de uma regra:


seletor{
    propriedade: valor; /* declaração */
}

   A regra é composta de um seletor (alvo da regra) e de uma declaração (parâmetros de estilo).

   Declaração, por sua vez, é composta por um par propriedade: valor

Exemplo de uma regra CSS

   Abaixo temos uma regra CSS que aplica três declarações:

  
p{                          /*seletor*/
    text-align: center; 
    color: yellow;          /*propriedades e declarações*/
    font-size: 20px;
}
  • Seletor: elemento p (regra aplicada a parágrafos)
  • Declarações: text-align: center; color: yellow; font-size: 20px;
  • Propriedades: text-align, color e font-size.

Tipos de Seletores

   Abaixo temos uma regra CSS que aplica três declarações:

  1. Seletor Universal: Representa todos os elementos. Simbolizado pelo *
    2.   
*{ color: green; } 
/*todos os elementos do documento terão a cor verde*/
  2. Seletor de Tipo (ou de Elemento): Representa um determinado elemento.
    3.   
p { color: red; } 
/* todos os parágrafos terão texto na cor vermelha */

h1 { color: yellow; } 
/* todos os elementos h1 terão texto em amarelo */
  3. Seletor de Classe: Aplica estilo a elementos com o atributo class definido.
    4.   
.verde { color: green; } 
/* Aptica cor verde aos e'ementos que possuam o atributo */
        
h1.verde { color: green; } 
/* Aplica cor verde apenas elementos h1 que possuam o atributo class= "verde" */
  4. Seletor de ID: Aplica estilo a elementos com o atributo id definido.
    5.   
#corAzul { color: blue; } 
/* Aplica cor azul ao elemento que possua o atributo id="corAzul" */
        
h1#corAzul { color: blue; } 
/* Aplica cor azul ao elemento h1 que possua o atributo id="corAzul" */

Convenções para nomes de id e classe

   Algumas convenções importantes para seguir ao criar nomes para atributos id e class:

  • Não usar espaços em branco
  • Não empregar caracteres especiais
  • CSS é case-sensitive
